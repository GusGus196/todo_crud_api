## ¿Qué es NodeJs?
- Qué es NodeJS? Usos

NodeJs es un entorno de ejecución de código abierto y multiplataforma para JavaScript, basado en el motor V8 de Chrome, que permite ejecutar JavaScript en el servidor fuera del navegador. NodeJs ejecuta JavaScript asíncrono basado en eventos, esta diseñado apra construir aplicaciones de red escalables. 

- Entorno de ejecución

NodeJs es un entorno de ejecución de código JS del lado del servidor, lo que en la práctica se refleja en la posibilidad de crear aplicaciones independientes en una máquina servidor, sin necesidad de un navegador

- Módulos

Los módulos en NodeJs son bloques de código reutilizables y encapsulados que permiten organizar, estructurar y mantener aplicaciones limpias al dividir funcionalidades especificas. Se clasifican en módulos integrados (core), locales y de terceros (instalados vía NPM), utilizando require() en CommonJs o import en ES Modules para su carga.

- Scripts

Los scripts en NodeJs permiten ejecutar código JavaScript fuera del navegador para tareas automatizadas, manipulación de archivos y backend.

- Event loop (a nivel conceptual)

Basicamente es un mecanismo central que permite la ejecución de operaciones asíncronas, no bloqueantes, incluso siendo Node.js un entono monohilo. Su función principal es gestionar efectivamente las entradas y salidas de alto rendimiento, revisando constantemente si el call stack está vacío para ejecutar callbacks pendientes.
